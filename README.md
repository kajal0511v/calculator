##Calculator 
A simple and responsive calculator build using HTML.CSS and Javascript.
This project perform basic operations such as addition,subtraction,multiplication,division and is suitable for beginners learning JavaScript fundamentals.

##Features
-Addition,subtraction,multiplication,division.
-Clear All(AC) button to reset calculations.
-User-friendly and clear UI.
-Responsive design for mobile and desktop.

##Technologies Used
-HTML
-CSS
-Javascript

#What I learned
-Handling button click events in Javascript.
-Using DOM methods to update the UI .
-Writing clean and reusable code.
-Styling layouts using CSS.

This project was created as a practice project to strengthen my basics of Javascript and Frontend Development.
